Detailed Description

This is an FFT library designed to run on ARM cortex M0 CPUs. It is based on FFT functions provided by ARM.

Adafruit invests time and resources providing this open source code, please support Adafruit and open-source hardware by purchasing products from Adafruit!

Written by Dean Miller for Adafruit Industries.

int ZeroFFT ( q15_t *  source,
uint16_t  length 
)

run an FFT on an int16_t array. Note that this is run in place.

Parameters
sourcethe data to FFT
lengththe length of the data. This must be a power of 2 and less than or equal to ZERO_FFT_MAX
Returns
0 on success, -1 on failure
Note
The FFT is run in place on the data. A hanning window is applied to the input data. The complex portion is discarded, and the real values are returned.
